# AIWED-ex-10

AI chatbot web app with:
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Model integration:** `meta-llama/Llama-3.1-8B-Instruct` via `llm.karunya.ai`

## Run Locally

```bash
cd Chatbot
npm install
cp .env.example .env
npm run dev:all
```

- Frontend: `http://127.0.0.1:5173`
- Backend: `http://127.0.0.1:8787`
